# 动态时间规整（DTW）距离

## 定义

动态时间规整（Dynamic Time Warping, DTW）是一种衡量两个时间序列相似度的算法，能够处理序列在时间轴上的非线性伸缩和偏移。

给定真值序列 $X = \{x_1, x_2, ..., x_n\}$ 和预测序列 $Y = \{y_1, y_2, ..., y_m\}$，DTW通过动态规划寻找最优对齐路径，计算两序列间的最小累积距离。

## 计算方法

构建累积成本矩阵 $D \in \mathbb{R}^{(n+1) \times (m+1)}$，初始化 $D(0,0) = 0$，其余元素为 $\infty$。对于每对元素 $(x_i, y_j)$，计算局部距离 $d(i,j) = (x_i - y_j)^2$，然后递推计算：

$$D(i,j) = d(i,j) + \min \begin{cases}
D(i-1,j) \\
D(i,j-1) \\
D(i-1,j-1)
\end{cases}, \quad i=1,...,n; \; j=1,...,m$$

最终DTW距离为：

$$\text{DTW}(X,Y) = \sqrt{D(n,m)}$$

归一化DTW距离为：

$$\text{DTW}_{\text{norm}}(X,Y) = \frac{\text{DTW}(X,Y)}{(n+m)/2}$$

## 特点

DTW距离越小表示序列越相似。相比欧氏距离，DTW能够容忍时间延迟和局部时间伸缩，更适合评估交通流预测中的趋势匹配性能。
