# 22stateEstimate.py 聚类功能说明

## 修改内容

已成功在 `src/22stateEstimate.py` 中添加了基于真值数据的速度聚类功能。

## 新增功能

### 1. 速度聚类函数 `cluster_speeds_and_get_thresholds()`

该函数对真值数据进行K-means聚类分析，将速度分为三类：
- **停车状态**（低速）
- **饱和流**（中速）
- **自由流**（高速）

#### 参数：
- `df`: DataFrame，包含node_id、time和速度列
- `node_id`: 节点ID
- `value_column`: 速度列名（默认为'avg_speed'）
- `start_frame`: 起始时间（可选）
- `end_frame`: 结束时间（可选）

#### 返回值：
字典包含以下内容：
- `stop_threshold`: 停车速度阈值（停车状态和饱和流之间的分界）
- `free_flow_threshold`: 自由流速度阈值（饱和流和自由流之间的分界）
- `cluster_centers`: 三个聚类中心（从低到高排序）
- `cluster_labels`: 每个数据点的聚类标签
- `speeds`: 原始速度数据

### 2. 运行流程

程序运行时会按以下步骤执行：

#### 步骤1：速度聚类分析
- 读取真值数据（默认为配置中的第一个文件）
- 对指定节点ID和时间范围内的速度数据进行K-means聚类
- 计算并打印两个关键阈值：
  - **停车速度阈值** (stop_threshold)
  - **自由流速度阈值** (free_flow_threshold)

#### 步骤2：交通状态估计
- 使用配置的阈值对所有数据文件进行交通状态估计
- 生成状态时间线图表

## 使用方法

直接运行脚本即可：

```bash
python src/22stateEstimate.py
```

## 可选配置

如果想使用聚类得到的阈值替代配置中的固定阈值，可以在 `main()` 函数中取消以下代码的注释：

```python
if cluster_result is not None:
    # 更新所有配置的阈值为聚类得到的阈值
    for config in data_configs:
        config['lower_threshold'] = cluster_result['stop_threshold']
        config['upper_threshold'] = cluster_result['free_flow_threshold']
```

## 技术细节

- 使用 `sklearn.cluster.KMeans` 进行聚类
- 聚类数量固定为3类
- 阈值计算方法：取相邻聚类中心的中点
- 自动过滤NaN值
- 支持指定时间范围进行聚类分析
